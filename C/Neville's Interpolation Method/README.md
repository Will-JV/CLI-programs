# Neville's Interpolation Method
Interpolates a function using Neville's algorithm

## Usage
Type the commands below in terminal
```bash
gcc filename.c -lm
```
```bash
./a.out
```
If gnuplot is not downloaded, type in the following command:
```bash
sudo apt install gnuplot
```
